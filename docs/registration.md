# Valid req On Registrations #


## *Post Request To /registrations* ##
> Credentials are sent through the headers
```json
{ 
    email: "example.com",
    password: "*****"
}
```

## *Facebook Users* ##
> Credentials are sent through the headers
```json
{
    Cookie: "_makeschool_session=*******",
    email: "example.com"
}
```
